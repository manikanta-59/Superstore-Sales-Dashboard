Superstore Sales Dashboard (Power BI)

 Project Overview
This Power BI dashboard analyzes sales performance of a retail superstore from 2019–2023.

It provides insights into:
- Total Sales, Profit, and Average Delivery Time
- Sales by Category and Subcategory
- Payment Mode distribution
- Regional and Segment-wise performance

 Tools Used
- Power BI Desktop
- Excel / CSV for data source
- DAX for calculated measures

Key Insights
- Technology category contributed **45% of total sales**
- Corporate segment generated **highest profit margins**
- Central region had **lowest YoY growth**

Files Included
- `Superstore_Sales_Dashboard.pbix` → Main Power BI dashboard file
- `Superstore_Dataset.csv` → Sample dataset
- `Insights.txt` → Detailed insights
